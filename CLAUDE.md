# 윤리경영 실천 Self-Check!

## 프로젝트 설명
윤리경영 실천을 스스로 점검하는 웹페이지입니다. 사용자가 윤리경영 관련 항목들을 체크하고 자신의 실천 수준을 확인할 수 있는 인터랙티브한 웹 애플리케이션입니다.

## 사용 기술
- **HTML**: 웹 페이지의 구조와 콘텐츠
- **CSS**: 스타일링 및 레이아웃
- **JavaScript**: 인터랙티브 기능 구현

## 파일 구조
```
/
├── index.html          # 메인 HTML 파일 (웹 페이지 구조)
├── css/
│   └── style.css       # 스타일시트 (디자인 및 레이아웃)
├── js/
│   └── main.js         # JavaScript 파일 (동작 및 기능)
└── CLAUDE.md           # 프로젝트 문서 (이 파일)
```

## 실행 방법
1. 프로젝트 폴더에서 `index.html` 파일을 찾습니다
2. 웹 브라우저(Chrome, Firefox, Safari 등)에서 `index.html` 파일을 엽니다
3. 또는 `index.html` 파일을 더블클릭하여 기본 브라우저에서 실행합니다

## 코딩 규칙

### 1. 주석 작성
- **모든 주석은 한국어로 작성**합니다
- 코드의 목적과 기능을 명확하게 설명합니다

### 2. 초보자 친화적 코드
- 복잡한 코드보다는 이해하기 쉬운 코드를 우선합니다
- 변수명과 함수명은 의미가 명확하도록 작성합니다
- 가능한 한 간단하고 직관적인 방법을 사용합니다

### 3. 주석 가이드라인
```javascript
// ✅ 좋은 예: 코드의 목적을 설명
// 사용자가 입력한 값을 검증하는 함수
function validateInput(value) {
    // 빈 값인지 확인
    if (value === '') {
        return false;
    }
    return true;
}
```

```html
<!-- ✅ HTML: 섹션의 역할 설명 -->
<!-- 윤리경영 체크리스트 영역 -->
<div class="checklist">
    <!-- 개별 체크 항목 -->
    <div class="item">...</div>
</div>
```

```css
/* ✅ CSS: 스타일의 목적 설명 */
/* 메인 컨테이너 - 중앙 정렬 및 최대 너비 설정 */
.container {
    max-width: 800px;
    margin: 0 auto;
}
```

### 4. 코드 작성 원칙
- 각 함수는 하나의 명확한 목적을 가집니다
- 매직 넘버 대신 의미 있는 변수명을 사용합니다
- 중요한 로직에는 반드시 설명 주석을 추가합니다
- 코드 블록마다 공백 라인으로 구분하여 가독성을 높입니다

## 개발 시 유의사항
- 모든 코드는 초보자도 이해할 수 있도록 작성합니다
- 새로운 기능 추가 시 반드시 주석으로 설명을 추가합니다
- 외부 라이브러리 사용을 최소화하고, 사용 시 그 이유를 주석으로 명시합니다
